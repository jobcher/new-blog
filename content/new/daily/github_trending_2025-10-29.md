#### 微软与 OpenAI 签署新协议，重塑长期合作关系
<p><b>微软与 OpenAI 签署新协议，重塑长期合作关系</b><br><br>2025年10月28日，微软宣布与 OpenAI 签署新的正式协议，进入合作伙伴关系的下一阶段。微软支持 OpenAI 组建公益型公司（Public Benefit Corporation, PBC）并完成资本重组。重组后，微软持有约1350亿美元的股份，折合约27%的稀释后股权比例（此前为32.5%）。<br><br>协议延续了双方在前沿模型和 Azure API 独家权的核心合作框架，直至 OpenAI 宣布实现通用人工智能（AGI）。AGI 的宣布将由独立专家小组验证。微软的模型和产品相关知识产权延长至2032年，并覆盖AGI之后的新模型；研究类IP权利保留至2030年或AGI确认前。微软的独家权不再包括 OpenAI 的消费硬件。<br><br>新条款允许 OpenAI 与第三方联合开发部分产品，其中 API 类产品仍专属 Azure。微软也可独立推进 AGI 研发。双方的收益分成机制维持至AGI确认，但付款周期延长。OpenAI 同时签约新增采购2500亿美元的 Azure 服务，微软不再拥有计算资源优先权。<br><br>此外，OpenAI 获准向美国国家安全部门提供API服务，并可在符合能力标准的情况下发布开放权重模型。<br><br>这一调整标志着微软与 OpenAI 的合作从战略绑定走向结构性分化，两家公司将在保持互信的前提下，获得更大自主空间以继续推动人工智能的发展。<br><br>来源：<a href="https://blogs.microsoft.com/blog/2025/10/28/the-next-chapter-of-the-microsoft-openai-partnership/" target="_blank" rel="noopener" onclick="return confirm('Open this link?\n\n'+this.href);">Microsoft 官方博客</a></p>
